Project Details:
Project Title: NoNet
Category: FinTech
Problem Statement: Public transport in regions like Nepal struggles with unreliable connectivity, cash-based fare collection, slow boarding, and lack of real-time tracking and transparency. A smart, connectivity-resilient system is needed to enable offline QR payments, live tracking, route guidance, and real-time revenue analytics for operators.
